# Imagini

Plasează aici fișierele de imagine. Lista necesară:

## Obligatorii (referențiate în index.html)
- `hero-aerial-night.jpg` — Image 2 din AI generation (twilight aerial cu lumina galbenă în recepție) — folosit ca hero principal
- `hero-aerial-day.jpg` — Image 1 (vedere aeriană la lumina zilei) — folosit în secțiunea "Despre proiect"
- `container-product.png` — Image 4 (single container product shot, fundal alb) — folosit în cele 3 box cards
- `container-interior.jpg` — Image 5 (interior container cu cutii) — folosit în secțiunea "Înăuntru"

## Opționale
- `entrance.jpg` — Image 3 (street view) — neutilizat în versiunea curentă, poți să-l adaugi la o galerie
- `og-cover.jpg` — 1200×630 pentru social sharing (Facebook, LinkedIn, Twitter cards). Crop din hero-aerial-night.
- `favicon.png` — 32×32 logo pentru tab browser
- `apple-touch-icon.png` — 180×180 pentru bookmark iOS

## Recomandare de optimizare
Înainte de deploy, optimizează toate JPG/PNG-urile pe:
- https://squoosh.app/ (recomandare: format MozJPEG, calitate 75-82)
- https://tinypng.com/ (drag & drop, automat)

Target:
- Hero images: max 350 KB la 1920×1080
- Product shots: max 200 KB
- Interior: max 250 KB
